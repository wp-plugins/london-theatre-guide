== Description ==

The 'London Theatre Guide' plugin will display a constantly updating list of theatre shows taking place in the capital. All of the big theatres are included, with everything from starry cast West End plays to stage musicals and Shakespearean tragedies. 

The listing will include a title, description and date for each show, alongside a picture of the poster. There are options to amend the date format, number of items displayed, and all of the CSS styles.

It has also been widgetized for drag-and-drop use on your sidebar.

== Installation ==

1. Upload the `london-theatre-guide` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu
3. Navigate to 'Settings > London Theatre Guide' to configure the plugin
4. You can either drag the `London Theatre Guide` widget straight onto your sidebar, or you can place `<?php london_theatre_shows(); ?>` into your template where you'd like the events to appear

== Upgrade ==

1. Delete the old `london-theatre-guide` folder
2. Upload the new `london-theatre-guide` folder to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu
4. Navigate to 'Settings > London Theatre Guide' to configure the plugin

== Frequently Asked Questions ==

= Where are the events taken from? =

The events are continually updated at the London Drum website -- http://www.londondrum.com/events/?cat=33

= How do I uninstall the plugin? =

Simply delete the old `london-theatre-guide` folder, and remove the `<?php london_theatre_shows(); ?>` line from your template file.
